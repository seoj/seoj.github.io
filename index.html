  • Force Directed Graph Drawing

    Implementation of Force-directed graph drawing algorithm.

  • Mandelbrot Set

    Implementation of Mandelbrot set with escape algorithm with basic black and white coloring. This implementation is SLLOOOOW, so be patient when the page seem to freeze. Click and drag the area you want to focus.

  • Maze

    A randomly generated maze. You can specify the width and height on the URL. Enabling smoothing makes it look nice but is MUCH slower. I recommend 50x50 as the maximum when using smoothing. If you turn off smoothing, you can go as large as 1000x1000.

    The algorithm uses a modified DFS where the tree is not pre-defined but randomly selected based on adjacency rules. The algorithm will attempt to create a maze where there is only 1 unique path from one point to another, and will attempt to use the maximum amount of space available.
